# n8n-nodes-read-pdf-with-password

This is an n8n community node. It lets you read a password-protected PDF in your n8n workflows.


[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/reference/license/) workflow automation platform.
